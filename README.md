### 1章 基本論理ゲートを用いたブール論理ゲートの作成

Mux,DMuxは基本論理ゲートを用いて実装

1入力nビット1出力1ビット論理ゲートについてはOr8wayを実装

多入力nビット多出力nビット論理ゲートについてはMux16-4way,Mux16-8way,DMux4way,DMux8wayを実装
